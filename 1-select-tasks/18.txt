Create a SQL query to calcualte total orders count by each day in 1998:

| Order Date | Total Number of Orders |
____________

SELECT CONVERT(date, [o].[OrderDate]) AS [Order Date], COUNT(*) AS [Total Number of Orders]
FROM [Northwind].[dbo].[Orders] [o]
WHERE datepart(YEAR, [o].[OrderDate]) = '1998'
GROUP BY CONVERT(date, [o].[OrderDate])
