Create a query to return:

| Customer Id | Total number of Orders | % of all orders |
 
order data by % - higher percent at the top.

--task4
SELECT [o].[CustomerID], COUNT([o].[CustomerID]) AS [Total number of Orders],
ROUND(CAST(COUNT([o].[OrderID]) AS NUMERIC) *100/SUM(COUNT([o].[OrderID])) OVER(), 3) AS [ % of all orders]
FROM [dbo].[Orders] [o]
GROUP BY [o].CustomerID
ORDER BY [ % of all orders ] DESC
