Create a query to return:

| Customer Id | Total number of Orders | % of all orders |
 
order data by % - higher percent at the top.

USE Northwind
;

SELECT CustomerID
, COUNT(*) AS TotalNumberOfOrders
, CAST ( COUNT(*) AS DECIMAL ) / SUM(COUNT(*)) OVER() * 100 AS [% of all orders]
FROM Northwind.dbo.Orders
GROUP BY CustomerID
ORDER BY CAST ( COUNT(*) AS DECIMAL ) / SUM(COUNT(*)) OVER() * 100 DESC
