Create a query to return all employees and full name of person to whom this employee reports to:

| EmployeeId | FullName | ReportsTo |

Full Name - title of courtesy with full name.
Order data by first column.
Reports To - Full name. If the employee does not report to anybody leave "-" in the column.
__________

SELECT [emp].[EmployeeID], [emp].[TitleOfCourtesy] + ' ' + [emp].[FirstName] + ' ' + [emp].[LastName] AS [FullName],
ISNULL([rep].[TitleOfCourtesy] + ' ' + [rep].[FirstName] + ' ' + [rep].[LastName], '-') AS [ReportsTo]
FROM [Northwind].[dbo].[Employees] [emp]
LEft JOIN [Northwind].[dbo].[Employees] [rep] ON [emp].[ReportsTo] = [rep].[EmployeeID]
ORDER BY [emp].[EmployeeID]
